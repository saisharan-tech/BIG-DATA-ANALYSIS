## PROJECT DESCRIPTION

The Netflix Data Analysis using PySpark project is developed as part of the Big Data Analysis internship task. The main objective of this project is to perform scalable data analysis on a large dataset using PySpark and derive meaningful insights.

In this project, a Netflix dataset containing information about movies and TV shows is analyzed using PySpark, which is a powerful big data processing framework. Unlike traditional tools like Pandas, PySpark is capable of handling large-scale datasets efficiently by distributing computations across multiple nodes.

The dataset includes features such as title, type (Movie/TV Show), director, cast, country, release year, rating, and duration. Using PySpark, the data is processed, cleaned, and analyzed to uncover trends and patterns in Netflix content.

## WORKING

The project begins by initializing a Spark session and loading the Netflix dataset into a PySpark DataFrame. Data preprocessing steps are performed to handle missing values and ensure proper formatting.

Using PySpark functions, various transformations and actions are applied to the dataset. Key operations include filtering data, grouping records, and performing aggregations to extract meaningful insights.

Some of the major analyses performed include:

- Distribution of Movies vs TV Shows  
- Content release trends over the years  
- Country-wise content production  
- Most common genres and ratings  
- Duration analysis of movies and shows  

The results are then visualized using Python libraries to make the insights more understandable and impactful.
